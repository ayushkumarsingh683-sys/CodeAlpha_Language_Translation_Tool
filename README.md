# 🌐 AI Language Translation Tool

An interactive, web-based language translation application built with Python. This project was developed as part of the Artificial Intelligence Internship at **CodeAlpha**.

---

## 🚀 Features
* **Multi-Language Support:** Translates text between major global languages.
* **Modern Web Interface:** Built with a clean UI using Streamlit.
* **AI Audio Playback:** Integrates Text-to-Speech (TTS) capabilities using gTTS.

---

## 🛠️ Tech Stack
* **Language:** Python
* **Frontend UI:** Streamlit
* **Translation Core:** Deep-Translator
* **Audio Engine:** gTTS
